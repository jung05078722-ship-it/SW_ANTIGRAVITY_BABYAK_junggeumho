1. 서비스 목적 
같은 학교 학생끼리 같이 밥 먹을 사람을 찾아주는 매칭 서비스이다. 학교 이메일로 인
증한 재학생만 사용할 수 있어서 안전하게 밥친구를 구할 수 있다. 
2. 대상 사용자 
혼밥이 잦은 대학생, 특히 신입생이나 편입생처럼 아직 학교 친구가 많지 않은 학생을 
주요 대상으로 한다. 
3. 핵심 기능 
① 회원가입 / 로그인 학교 이메일로 인증 코드를 받아 가입한다. 닉네임, 학과, 학년, 음
식 취향 태그를 설정한다. 
• 이메일 입력 → 인증 코드 발송 → 코드 확인 → 프로필 설정 
② 밥약 글 올리기 날짜, 시간, 장소, 인원을 설정해서 모집 글을 게시한다. 
• 날짜·시간 선택 → 장소 선택(학교 식당 또는 직접 입력) → 인원·성별·메뉴 조건 
설정 → 게시 
③ 글 목록 보기 & 신청하기 메인 화면에서 모집 중인 글을 카드 형태로 보고 필터(날
짜, 시간대, 메뉴)로 걸러볼 수 있다. 마음에 드는 글에 신청 버튼을 누르면 글 작성자에
게 알림이 간다. 
• 목록 탐색 → 상세 보기 → 신청 버튼 탭 → 작성자 수락/거절 → 수락 시 채팅방 
열림 
④ 그룹 채팅 수락된 멤버들끼리 채팅방에서 세부 약속을 조율한다. 식사 후 채팅방이 
종료되면 리뷰 작성 화면이 뜬다.
